== Overview

This workshop provides hands-on exercises for Red Hat Advanced Cluster Management (RHACM) {acm_version} on OpenShift {ocp_version}.

You will learn to manage a multicluster fleet, deploy applications with ArgoCD, enforce governance policies, and integrate AI-powered operations -- all from a single ACM hub.

*Target versions:* OpenShift {ocp_version} / RHACM {acm_version}

== Base Environment

This workshop was built on top of the https://catalog.demo.redhat.com/catalog?item=babylon-catalog-prod/published.ocp4-acm-demo.prod&utm_source=webapp&utm_medium=share-link[Advanced Cluster Management for Kubernetes Demo] from the Red Hat Demo Platform. Order this catalog item to get a pre-configured hub cluster with ACM already installed, then follow the modules below.

== Prerequisites

* A running OpenShift {ocp_version}+ cluster with RHACM {acm_version} installed (see *Base Environment* above)
* The `oc` CLI tool installed
* The `kubectl` CLI tool installed
* The `git` CLI tool installed
* A GitHub account
* AWS credentials (for provisioning managed clusters via Hive)

== Quick Start

Run the setup validation script to confirm your environment is ready:

[source,bash]
----
./setup.sh
----

== Workshop Architecture

[source]
----
Hub Cluster (OCP 4.20, ACM 2.15, SNO)
  ├── local-cluster   (self-managed, labels: environment=hub)
  ├── standard-cluster (Hive AWS SNO, labels: environment=dev)
  └── gpu-cluster      (Hive AWS SNO g6.4xlarge, labels: gpu=true, accelerator=nvidia-l4)
----
